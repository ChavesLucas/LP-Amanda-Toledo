# Landing Page - Dra Amanda Toledo

## Arquivos Inclusos

- `index.html` - Página principal
- `styles.css` - Estilos CSS
- `script.js` - JavaScript
- `.htaccess` - Configurações do servidor
- `README.md` - Este arquivo

## Como Instalar no Hostinger

### Passo 1: Preparar os Arquivos

1. Baixe os 5 arquivos
2. Certifique-se de que estão descompactados

### Passo 2: Fazer Upload no Hostinger

1. Acesse o **Gerenciador de Arquivos** do Hostinger
2. Navegue até a pasta **`public_html/`**
3. **Importante:** Não crie subpastas! Faça upload dos 5 arquivos DIRETAMENTE em `public_html/`:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.htaccess`
   - `README.md`

### Passo 3: Configurar o Domínio

1. No painel Hostinger, vá para **Domínios**
2. Selecione seu domínio
3. Em **Apontamento de Domínio**, certifique-se de que está apontando para `public_html/`
4. Aguarde a propagação do DNS (pode levar até 24 horas)

### Passo 4: Testar

1. Acesse seu domínio no navegador
2. Verifique se a página carrega corretamente
3. Teste os botões de WhatsApp e links internos

## Suporte

Se tiver problemas:

1. Verifique se os arquivos estão em `public_html/` (não em subpasta)
2. Limpe o cache do navegador (Ctrl+Shift+Delete)
3. Aguarde a propagação do DNS (até 24 horas)
4. Verifique o console do navegador (F12) para erros
